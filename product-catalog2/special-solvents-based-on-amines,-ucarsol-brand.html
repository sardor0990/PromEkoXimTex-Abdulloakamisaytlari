Специальные сольвенты на основе аминов марки Ucarsol

Описание Область применения

Сольвент UCARSOL HS 101 для очистки газа
   Широкий диапазон применения в области очистки газа, например на нефтеперерабатывающих и нефтехимических заводах, при очистке природных и отходящих газов.

Сольвент UCARSOL HS 102,103 для селективного удаления H2S
    Этот продукт был специально разработан для очистки газовых потоков низкого и высокого давления, требующих высокой селективности. 

Сольвент UCARSOL HS 104 для удаления кислого газа
   Сольвент UCARSOL HS 104 специально разработан для селективного удаления H2S из потоков углеводородов, содержащих как СО2, так и H2S.

Сольвент UCARSOL HS 115 для удаления кислого газа
   Сольвент UCARSOL HS 115, представляющий собой значительный прогресс в области удаления кислого газа, специально разработан с целью уменьшения потерь амина при обработке жидких углеводородов. 

Сольвент UCARSOL LE 713 для очистки газа при нефтепереработке и сжиженных газов.

Сольвент UCARSOL NH 608 для очистки конвертированного газа при производстве аммиака
   Специально разработанный для удаления двуокиси углерода (СО2) на установках синтеза аммиака. Сольвент UCARSOL NH 608 обеспечивает значительную экономию эксплуатационных затрат по сравнению с растворителями, обычно используемыми на установках обработки аммиака.

UCARSOL HS

-Сольвенты для селективной очистки кислых газов в условиях широкого спектра задач,включая сжиженный нефтяной газ.

UCARSOL LE

-Сольвенты для газа нефтепереработки и сжиженного нефтянного газа. Очистка топливных газов, газов рециркуляции и/или крекинг-газа.

UCARSOL NH

-Сольвенты для очистки агрессивных кислых газов, позволяющий существенно снизить эксплутационные расходы по сравнению с обычными аминами.
